HTML IN MY STYLE 

    
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
BASIC EXAMPLE :
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
<DOCTYPE html >
<html>
<head>
<title>HTML IN MY STYLE</title >
</head>
<body>
<h1>THIS IS MY FIRST HTML PROGRAM< >
<p>THIS IS MY FIRST PARAGRAPH< >
</body>
</html>
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
OUTPUT :
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
DEFINATIONS :
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. !DOCTYPE html: Declares the document type and version of HTML being used (HTML5 in this case).
2. html: The root element that contains all the content and structure of the HTML document.
3. head: Contains metadata about the document, like its title, character set, and linked resources.
4. title: Defines the title of the webpage, which is displayed in the browser's title bar or tab.
5. body: Contains the visible content of the webpage, like text, images, and other elements.
6. h1: Represents the highest-level heading, usually used for main titles or important headings.
7. p: Defines a paragraph of text.
